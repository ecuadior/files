Machine Learning

On-Device Neural
Network Inference

A 314-parameter feedforward neural network — trained in Python, exported as a C++ header, and running entirely on the Xiao ESP32-C6 with no external library.

Why Upgrade from Thresholds?
⚠ THRESHOLD DETECTION (CURRENT)
Requires manual re-tuning for every user — hand size and glove fit vary session to session
Only REST and PINCH — adding more gestures means more hardcoded if/else rules
Binary output — finger is either bent or not, no nuance for proportional control
Breaks if glove shifts even slightly during a session
PINCH_THRESHOLD is a magic number from trial and error
✓ NEURAL NETWORK (ML UPGRADE)
Learns user-specific patterns — generalizes to new sessions automatically
Scales to any number of gestures by adding training data and one constant
Continuous confidence score (0–100%) enables smooth, proportional arm control
Captures temporal patterns across 1.5s window — robust to glove drift
Validated on held-out data — known, measurable accuracy metric

📐

9 Input Features

3 features × 3 channels extracted over a 15-sample (1.5 second) sliding window: mean (sustained level), std deviation (fluctuation / activity), slope (is the finger moving?).

⚡

314 Parameters

9×16+16 + 16×8+8 + 8×2+2 = 314 total weights and biases. Flash footprint: ~1.3 KB. Inference time: <1 ms at 160 MHz RISC-V.

🎯

Softmax Output

P(PINCH) = 0.87 means 87% confident it's a pinch. This value is used directly as pinch intensity — squeezing harder produces more extreme features and higher confidence.

Training Pipeline

5-Step Process: Data to Device

📋

1 — Collect Data (MATLAB)

Run NumGesturecollector.m. Sound cue → hold REST for 10s → CSV saved. Repeat for PINCH. Aim for 3–5 sessions per gesture at different wrist angles.

🔬

2 — Extract Features (Python)

Sliding window (size=15, step=5). Per channel: compute mean, std, linear slope. Output: 9 features per window. Apply StandardScaler normalization.

🏋️

3 — Train (TensorFlow/Keras)

Adam optimizer, lr=0.005. Up to 120 epochs with EarlyStopping. 80/20 train/test split. Target: >85% val accuracy. Current model: 100% on collected data.

📄

4 — Export Weights (C++ Header)

All 314 weights + biases + normalization params written to gesture_model_weights.h. No TFLite needed — forward pass is just float multiplications.

📟

5 — Deploy to Xiao

Copy .h file into GloveDemoML/ folder. Flash GloveDemoML.ino. Window fills after 1.5s (~15 samples). Inference every 0.5s. Output: NN: PINCH conf=0.87
