FALL 2025 — EENG 489

Project Proposal & Initial Research ✓ Done

Team formation, project scope definition, literature review on inductive sensing and gesture-controlled robotics. Initial component selection and budget planning.

Submitted capstone proposal — Smart Glove for pharmaceutical lab remote control
Identified LDC1614EVM as the inductance-to-digital converter
Defined 4 target gestures: Rest, Pinch, Flexion/Extension, Adduction
Selected Seeed Xiao ESP32-C6 for both glove and arm microcontrollers
FALL 2025 — EENG 489

Glove Fabrication & Sensor Design ✓ Done

Construction of the physical glove with sewn conductive thread coils. Sensor placement based on anatomical joint positions.

5 inductive coil sensors sewn: thumb, index, middle, ulnar, wrist crease
Coil design: 3–5 loops per sensor, ~2mm loop spacing for max inductance
Sensor placement spans from MCP to DIP joints for full range motion capture
Tested single-thread vs. stitched multi-segment coils for signal integrity
FALL 2025 / SPRING 2026

LDC1614 I2C Driver & PCB Integration ✓ Done

Wrote the low-level I2C driver for the LDC1614EVM. Configured all 4 channels and debugged the CONFIG register issue that caused all-zero outputs.

Identified critical CONFIG bug: 0x16 vs 0x14 — solved all-zero channel issue
All 4 channels reading in 59–63 million counts range at rest
I2C address confirmed at 0x2A — ADDR pin tied internally to GND on EVM
Wiring standard: D4/GPIO21 = SDA (blue), D5/GPIO23 = SCL (green)
SPRING 2026 — Week 7

Data Collection & Calibration ✓ Done

Systematic collection of REST and PINCH gesture data using sound-cued sessions. Inductor configuration optimized for clean signals.

Tested single 470µH inductor per finger → switched to two in series for cleaner data
REST baselines: Thumb ~4.231MHz, Index ~1.470MHz, Middle ~28.79MHz
10 sessions × 10 seconds each for both REST and PINCH gestures
Normalized pinch values: Thumb −0.507, Index −2.060, Middle −2.446
SPRING 2026

ML Pipeline & Neural Network Training ✓ Done

Built complete Python training pipeline. Trained 9→16→8→2 feedforward network. Exported weights as C++ header for on-device inference.

Feature extraction: mean, std deviation, slope per channel over 15-sample window
Network: 314 parameters, ~1.3KB Flash, <1ms inference at 160MHz
Achieved 100% training accuracy on collected gesture data
Weights exported to gesture_model_weights.h for GloveDemoML.ino
SPRING 2026

ESP-NOW Wireless Link & Arm Receiver ✓ Done

Implemented wireless communication. Arm receiver controls servos based on received gesture packets.

ESP-NOW protocol: 11-byte packet, <5ms latency
ArmReceiver.ino: startup servo self-test, smooth proportional servo control
New build: PCA9685 I2C servo driver replaces direct GPIO for expansion
Python virtual dashboard reads machine-readable serial output for presentation
SPRING 2026 — NOW

Proof-of-Concept Demo Preparation ⏳ In Progress

Assembling the 4-stage demo guide. Verifying all hardware connections, firmware stability, and Python dashboard on demo machine.

4-stage demo: Sensor health check → MATLAB visualization → gesture detection → wireless pipeline
Single-Xiao fallback mode ready if second board unavailable during demo
Demo talk track prepared for judges — 5 key talking points
SPRING 2026 — FINAL

Final Prototype & Presentation Upcoming

Full physical robotic arm assembly with 3 servos. End-to-end live demonstration. Final written report and oral presentation to faculty and judges.

3× SG90/MG90S servos — one per finger of the gripper
Per-finger independent control by expanding ESP-NOW packet
Final report with ML validation accuracy and system architecture
