Step 1 — Calibration

Sensor Baseline & Spread Values

Calibration establishes a baseline and spread from the resting (open-hand) position. The baseline is the average sensor reading at rest. The spread quantifies the natural variation in signal while the hand is stationary — caused by electrical noise and minor involuntary muscle movement. Both values are required before normalization can occur.

Step 3 — Normalization

Normalizing Sensor Readings

Raw frequency values differ across sensors and users. Normalization converts each reading into a standardized value representing how far it deviates from the resting state in units of spread. This makes the classifier robust to differences between users and sessions.

Normalization Equation
Xnorm = ( X − b ) / s
Xnorm — normalized sensor value
X — current sensor reading
b — baseline (resting average)
s — spread (natural variation at rest)
Why normalized pinch values are negative: Any decrease in frequency — such as when a finger bends — results in a negative normalized value after subtracting the baseline. The steep drop into the negative range is expected and confirms correct behavior.

Step 4 — Pinch Intensity

Proportional Gesture Intensity

Pinch intensity represents how strongly the user is performing the pinch gesture. A larger value = stronger pinch = robotic fingers bend more. This enables smooth, proportional servo control rather than simple on/off behavior.

Pinch Intensity Equation
intensity = value / max-pinch
value — current normalized sensor reading
max-pinch — maximum observed pinch value per finger
