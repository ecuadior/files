Stage-by-Stage Walkthrough
1
~15 MIN · SENSOR HEALTH CHECK

Verify All Channels Are Live

Flash existing LDC firmware. Call RawData() in loop(). Open Serial Monitor at 115200 baud. Verify no channel shows 0, 4294967295, or 268435455 constantly. Bend each finger individually — only that channel should respond. This confirms sensor independence and correct wiring before anything else.

// Expected Serial Monitor output: Channel 0: 14523891 Channel 1: 11234567 Channel 2: 11892034 Channel 3: 12100456
2
~10 MIN · MATLAB LIVE VISUALIZATION

Show Real-Time Inductance Curves

Switch loop() to call MATLAB(). Run NUMChannelDataCollector.m — set port and numChannels=3. Three live subplots appear. Bend each finger one at a time and demonstrate that only that channel's trace moves. Optionally save REST and PINCH CSV files with NumGesturecollector.m as training data artifacts to show judges.

Close the Arduino Serial Monitor before running MATLAB. Only one program can own the COM port at a time. MATLAB will show "Port busy" otherwise.
3
~15 MIN · ON-DEVICE GESTURE DETECTION

Live Classification on the Glove Xiao

Flash GloveDemo.ino. On startup, calibration runs for 6 seconds — hold hand completely open and flat. After calibration, Serial Monitor shows classification at 10Hz. If PINCH isn't detected when squeezing: check delta values and lower PINCH_THRESHOLD. If REST shows false PINCH: raise threshold or recalibrate.

62963745,59838582,59853714 Thumb:62963745 Index:59838582 Middle:59853714 | GESTURE: REST ← pinch fingers → Thumb:65000000 Index:61900000 Middle:62500000 | GESTURE: PINCH (72%)
4
~15 MIN · WIRELESS END-TO-END + VIRTUAL ARM

Full Pipeline — Two Xiaos, Python Dashboard

Flash ArmReceiver.ino to second Xiao. Note its MAC address from Serial Monitor. Update ARM_XIAO_MAC[] in GloveDemo.ino with that exact MAC. Reflash glove Xiao. Run Python dashboard. Pinch and watch GRIPPER angle move. LED blinks on every packet received. Packet count in dashboard increments continuously.

// Run the Python dashboard: python virtual_arm_dashboard.py --port COM6 --source arm // Single-Xiao fallback: python virtual_arm_dashboard.py --port COM5 --source glove
Single-Xiao fallback: If only one Xiao is available, run the dashboard directly from the glove Xiao with --source glove. The Python script does gesture detection locally from the CSV stream.
Demo Talk Track

5 Talking Points for Judges

#ShowSay
1 Point to glove + LDC board Stainless steel inductive threads sewn into each finger. As a finger bends, the inductance changes. The LDC1614 converts those changes into a 28-bit number 10 times per second.
2 MATLAB plot — bend each finger Each subplot is one finger. When I bend just the index, only that channel moves — the others stay flat. That proves our sensors are independent and correctly wired.
3 Serial Monitor — REST vs PINCH The last word on each line is the classified gesture. Open hand — REST. Pinch — PINCH, with percent intensity showing how hard I'm squeezing.
4 Python dashboard — perform gestures The second Xiao is receiving the gesture wirelessly over ESP-NOW — packet count is incrementing — and it maps the signal to servo angles. GRIPPER is partially closed. Squeeze harder — now it's at 0 degrees.
5 Show two CSV files from MATLAB These are the raw inductance readings for REST and PINCH. The values are clearly separated, which is exactly what our ML classifier needs to generalize beyond simple thresholds.
